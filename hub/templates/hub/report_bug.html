Reporting Guidelines
Before Reporting:
  • Clear your browser cache
  • Try refreshing the page
  • Check your internet connection
  • Try a different browser
What to Include:
  • Detailed description
  • Step-by-step reproduction
  • Expected vs actual behavior
  • Browser and device info
  • Screenshots if possible
Response Time
We typically respond within:
  • Critical bugs: 2-4 hours
  • High priority: 24 hours
  • Medium priority: 2-3 days
  • Low priority: 1 week

Common Issues & Solutions

Search Not Working

If the search function isn't working properly, try:

  • Clearing your browser cache
  • Using different search terms
  • Checking your internet connection
Images Not Loading

If images aren't displaying:

  • Refresh the page
  • Check your internet speed
  • Try a different browser
  • Clear browser cache
Cart Issues

If you're having cart problems:

  • Log out and log back in
  • Clear browser cookies
  • Try adding items again
  • Check if you're logged in
Mobile Issues

For mobile-specific problems:

  • Update your mobile browser
  • Try desktop version
  • Check mobile data connection
  • Restart your device
